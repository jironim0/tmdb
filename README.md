# 🎬 TMDB Movie App

Веб-приложение для поиска фильмов и просмотра информации о них, использующее [The Movie Database (TMDB) API](https://www.themoviedb.org).

## 🚀 Демо
Посмотреть проект вживую: https://tmdb-brown.vercel.app/

## 🛠 Стек технологий
- **React** (Vite)
- **Redux Toolkit** (RTK Query для запросов к API)
- **TypeScript**
- **CSS Modules**
- **React Hot Toast**

## ⚙️ Функционал
- [x] Просмотр списка популярных и новых фильмов
- [x] Детальная страница фильма
- [x] Поиск по названию
- [x] Фильтрация по жанрам и рейтингу
- [x] Добавление в «Избранное» (хранится в LocalStorage/Redux)
